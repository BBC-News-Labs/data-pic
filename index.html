What is an h2?
The h1 to h6 tags are used to define HTML headings.
h1 defines the most important heading. h6 defines the least important heading.
Why do we use a h3 here?
The h1 tags can only be used once on a page. But the others, like h3 can be used multiple times on a webpage as subheadings for the page's sub topics.
What is a paragraph tag?
The p element defines a paragraph, so browsers will automatically add some space before and after it. We have added styles to make the impact figure more prominent as it is the most important piece of data.
Why do the impact figure and data description sit in the same div?
A div tag defines a division or a section in an HTML document. The div tag is used to group block-elements so they can be styled together by the CSS code. The impact figure and its description are part of the same div as they relate to one another.
Whats is an unordered list?
An unordered list starts with the ul tag. Each list item starts with the li tag. A list can contain as many items as you like.
